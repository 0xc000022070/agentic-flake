# examples

Minimal `mkSkill` examples for real project layouts:

- [`local-project/flake.nix`](./local-project/flake.nix): local `./skills` directory
- [`github-encoredev/flake.nix`](./github-encoredev/flake.nix): fetched GitHub repo where the repo root is the skills root
- [`github-openfang/flake.nix`](./github-openfang/flake.nix): fetched GitHub repo where skills live in a nested subdirectory
